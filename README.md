# snake_game
Text based simple snake game written in python

> python main.py
```
_ _ _ _ _ _ _ _ 
 | | | | | | | |
_ _ _ _ _ _ _ _ 
 | | | | |@| | |
_ _ _ _ _ _ _ _ 
 | | | | |@|@|>|
_ _ _ _ _ _ _ _ 
 | | | |*| | | |
_ _ _ _ _ _ _ _ 
 | | | | | | | |
_ _ _ _ _ _ _ _ 
 | | | | | | | |

```
Use arrow keys to navigate the snake. If you press any key other than arrow keys the game won't run correctly.
